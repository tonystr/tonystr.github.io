ATIRAG release!

21. Mar 2018


Here it is! Development of ATIRAG has concluded, and it's now avalible for public use. The game is relatively simplistic, and most gamedevs could make better games in less than a week, but I have made important progress with myself in regards to time management and actual game development. Five years have gone since I picked up Gamemaker, and not a single game has come of it (except one for a game jam). This game marks an important milestone for me and my capability of completing a game. I did not manage to add all the features I wanted within the time limit, though this further contributed to a better understanding of time and how to use it. My next project will not have a completion-deadline set from the very start, but after a week or so, it will get a beta-deadline, in which I release an unfinnished version of my game, in order to get some feedback along the way.

Here are some gifs of the finished game! Link to download can be found at the bottom of the page.

So what did I actually change/add this last week? Well, first of all, the game now has a menu! A rather stylish one, I might add. There are actually three menus; the main menu, the pause menu and the game over menu. The pause menu had the most work put into it, while the game over menu had the least. Because of the creeping deadline, I somewhat rushed these menus, therefore the code is very messy, and they're not very flexible, but the end result was pretty enough for me to be satisfied.

Here is a gif of the menus. (The mouse is offset because of the recording software)

ATIRAG 2018

I also added a highscore feature which is automatically saved upon death. The current score is drawn on top of the player, as that simply felt the most natural. The abstract artstyle of the game helped this look decent. Additionally, there have been minor balancing changes, a couple bug-fixes, and various tweaks to code that probably isn't too visible in the game.

Anyway, enough talking. Hop over to TonyStr.net/.../atirag and grab yourself a free copy!
