UpDownDuckAround

This is a video game created in 72 hours. It was made for the GMC gamejam #4. With the limit of time, I had to sacrifice graphics for gameplay. The theme for the jam was "Two Sides Of The Same Coin", which I incorpriated through splitting the same game into two, different games. The objective is the same on both sides: to survive. You loose by hitting either the top or bottom of the screen, but I soon realized this was not challenging enough, even with speeding up the platforms over time, so I added red projectiles to spice the game up.

Post-release feedback has described the game as "brutally hard".
